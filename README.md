# PythonBrincadeirasDados
A ideia desse projeto é ir testando estatistica e ideias de implementação estatistica com alguns dados abertos ao publicos

## Analise_terceiras
  Aqui tem algum passo a passo e alguns incrementos de alguma análise já feita por alguma outra pessoa disponivel em git, youtube, blog, aula, etc.


## CounterStrike
  Possui algumas bases de dados e estátistica de jogos no geral, a ideia é posteriormente fazer uma brincadeira de quando ocorre alguma transferência, analisar o jogador que foi transferido e se isso vai interferir positivamente ou negativamente no time.

  - [x] Atualmente possui uma análise de ganhos do time da Team Liquid no Counter-Strike, League of Legends, DotA 2 e Rainbow 6. 
  Como o dota possui o The International, com uma alta premiação independente da colocação, que pode afetar as analises, foi feito também um conjunto de dados sem esse torneio se preferir.
  
  -> Um dashboard interativo está disponível em: https://team-liquid-earnings.streamlit.app/
