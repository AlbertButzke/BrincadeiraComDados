# Python: Brincadeiras e Dados 📊
Este repositório é um laboratório pessoal de Ciência de Dados e Estatística. O objetivo é explorar datasets públicos, testar implementações estatísticas e aprimorar habilidades de visualização e modelagem preditiva.

## 🚀 Projetos em Destaque
### 🎮 Counter-Strike & E-sports Analytics
Análise de performance e ganhos financeiros de organizações de e-sports, com foco inicial na **Team Liquid**.

 - **Status**: Análise de premiações concluída (CS, LoL, Dota 2 e R6).

 - **Destaque**: Tratamento especial para dados do *The International* (Dota 2) para evitar vieses estatísticos devido às premiações atípicas.

 - **Live Demo**: Visualize o [Dashboard Interativo](https://team-liquid-earnings.streamlit.app/)

 - **Próximos Passos**: Modelagem de impacto de transferências de jogadores no desempenho da equipe.

### 💳 Detecção de Fraude e Crédito
Estudo focado em transações bancárias e análise de risco.

 - **Técnicas**: Exploração de datasets de crédito para identificação de padrões e comportamento transacional.

 ### 🚀 IBM & SpaceX: Predição de Pousos
Projeto desenvolvido durante a certificação da IBM via Coursera.

 - **Foco**: Limpeza de dados, EDA (Exploratory Data Analysis) e modelos de classificação para prever o sucesso de pousos do primeiro estágio do Falcon 9.

## 🧪 Laboratório de Estudos (Work in Progress)
 - **Analise_terceiras**: Repositório de "estudos de caso". Aqui eu replico e incremento análises de referências da comunidade (GitHub, YouTube, blogs técnicos), adicionando novas camadas estatísticas.

 - **World Cup 2018*: Projeto planejado para focar em tratamento intensivo de dados históricos e análise de performance em grandes torneios mundiais.

## 🛠️ Toolbox Técnica
 - **Linguagens**: Python (NumPy, Pandas, Scikit-Learn)

 - **Estatística**: Inferência, análise de distribuição e métodos de reamostragem.

 - **Visualização**: Matplotlib, Seaborn, Plotly e Streamlit para dashboards.

 - **Documentação**: LaTeX para formulações matemáticas complexas.
